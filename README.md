# Auren

Auren is a minimal Expo + React Native + TypeScript personal AI app prototype.

## Start

```bash
npm install
npm run start
```

Then open the app with Expo Go or an iOS/Android simulator.

## Scripts

```bash
npm run start
npm run typecheck
```

## Current scope

- Expo app shell
- React Native + TypeScript setup
- Minimal Auren home screen
- Premium floating composer UI
- Reusable theme, icons, action pill, and composer components
